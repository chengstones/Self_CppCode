## 线性表
### 顺序表
- 扩充操作的时候doublespace()函数需要maxsize*=2
- insert 操作需要从后往前，即
```cpp
for(int j(currentLength);j>i;--j) elem[j]=elem[j-1];
```
- remove 操作从前往后, 即for(int j(i);
```cpp
j<currentLength-1;++j) elem[j] = elem[j+1]
```
### 链表
- 清空操作中delete head->next 后 需要将head->next = NULL
